# Company Discovery Agent

Turn a company name or website into one clean, deduplicated account in your workspace. Hand the agent whatever you know — a name, a domain, or both — and it returns the matching account if one already exists, or creates a fresh record when it does not. When Apollo is connected, it tags the account with an Apollo organization reference so downstream enrichment has something to bind to.

## Works with

- Apollo

## Capabilities

- Resolve a company name or website into a single canonical account
- Reuse an existing account when the same company is already in your workspace
- Create a new account when the company is genuinely new
- Tag accounts with an Apollo organization reference when Apollo is connected
- Run safely in batch as a building block inside larger prospecting flows
